# SyntaxVoid UI Design System

## Tokens
Defined in `packages/syntaxvoid-ui-kit/styles/tokens.less`.
Prefix: `--sv-*`

## Skins
- **Clean** (Default): Modern, flat, rounded.
- **Pixel**: Retro, sharp edges, scanlines (optional).

## Usage
Import `syntaxvoid-ui-kit` in your package.

## Components
- `.sv-panel`
- `.sv-header`
- `.sv-body`
- `.sv-footer`
- `.sv-btn`
- `.sv-input`
- `.sv-badge`

## Configuration
Set `syntaxvoid.ui.skin` to `'clean'` or `'pixel'` in your `config.cson` or via package settings.
